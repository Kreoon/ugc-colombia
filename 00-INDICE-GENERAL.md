## Propósito

Esta carpeta contiene cinco packs operativos en formato PDF-ready (Markdown) diseñados para que cada miembro del equipo tenga un único documento de referencia con todo lo que necesita para ejecutar su rol. Todos los packs están consolidados desde el repositorio fuente de UGC Colombia (carpeta `content/`) con fecha de corte 2026-04-08.

**Regla de uso:** si hay contradicción entre este PDF y el repositorio GitHub, gana el repositorio (es la fuente viva). Estos PDFs son snapshot operativo para lectura offline en Drive.

## Packs disponibles

### 01-PACK-ALEXANDER-CEO.md
**Dirigido a:** Alexander Cast (Founder / CEO)
**Contenido:** Resumen ejecutivo del negocio, OKRs Q2-Q3 2026, decisiones pendientes que bloquean Q2, script de discovery call, plan de contenido 90 días en LinkedIn + multicanal, playbook de renewal call día 90.
**Uso:** Documento de cabecera para revisiones semanales y mensuales. Consultar antes de cada call comercial importante y renewal.

### 02-PACK-DIANA-CREATORS.md
**Dirigido a:** Diana Mile (Head of Operations / Creators)
**Contenido:** Sistema operativo completo de creadores (8 documentos integrados) — formulario de aplicación, scoring A/B/C, contrato, pricing guide, briefs por formato, proceso de revisión y feedback, dashboard de métricas. Plus: flujo de onboarding de cliente día 0-90 (etapas 1-10) y playbook de cliente difícil.
**Uso:** Biblia operativa diaria. Usar como referencia en cada kickoff, cada brief, cada escalamiento con creador o cliente.

### 03-PACK-BRIAN-FINANCE.md
**Dirigido a:** Brian (Finanzas)
**Contenido:** Pricing guide LATAM/USA por formato, condiciones de pago, proceso de facturación al cliente (cobro 50/50 + cobranza), proceso de pago a creadores con idempotencia, cálculo de MRR / Health Score / DSO / margen bruto, política de cobranza, riesgos financieros y mitigación.
**Uso:** Ejecutar cada billing cycle, preparar MBR mensual, gestionar facturas vencidas y health score de clientes.

### 04-PACK-TANYA-COMMUNITY.md
**Dirigido a:** Tanya (Community & Content)
**Contenido:** Plan completo de comunidad de creadores (Discord + WhatsApp, roles, gamificación, onboarding día 0-30), calendario editorial de las 7 redes sociales, banco de hooks + guiones de referencia, flows de WhatsApp Business.
**Uso:** Operación diaria del pool de creadores (capa comunidad) + publicaciones semanales + atención L1 por WhatsApp.

### 05-PACK-SAMUEL-TECH.md
**Dirigido a:** Samuel (Tech Lead)
**Contenido:** Arquitectura AI de ugccolombia.co (Next.js 15 + Supabase + AI Gateway), schema Supabase completo, features AI (chatbot, dossier, briefs, match de creadores, reportes), especificación de integraciones API, workflows n8n relevantes, hallazgos del security audit y fixes pendientes, variables de entorno necesarias.
**Uso:** Desarrollo diario del stack, referencia para implementar migraciones, setup de entorno nuevo, auditoría de seguridad.
