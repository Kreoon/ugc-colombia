# Presentation Kit Spec — UGC Colombia

> Deck comercial de 16 slides para venta B2B.
> Audiencia: Founder D2C LATAM, Brand Manager USA Hispanic, Agencia de performance.
> Formato: Google Slides 16:9 (1920×1080px) / Canva Pro.
> Sistema visual: Negro + Dorado + Crema, Fraunces + Inter.

---

## Configuración del Deck

```
Formato:          Widescreen 16:9 (1920 × 1080px)
Fondo base:       #0A0A0A (negro editorial)
Fuente principal: Fraunces (titulares) / Inter (body)
Idiomas:          Español neutro (con opción EN para USA clients)
Longitud:         16 slides (sin apéndice) + slides de soporte opcionales
Duración oral:    15–20 minutos para presentación completa
```

## Estilo Visual General del Deck

### Reglas de composición por tipo de slide

| Tipo              | Fondo          | Acento       | Imagen                         |
|-------------------|----------------|--------------|--------------------------------|
| Cover / Cierre    | #0A0A0A        | #FFD60A      | Full-bleed con overlay negro   |
| Pain / Diagnóstico| #0A0A0A        | #DC2626      | Mínima o ninguna               |
| Proceso / Cómo    | #111111        | #FFD60A      | Iconografía outline dorada     |
| Proof / Casos     | #0A0A0A        | #FFD60A      | Screenshot real Kreoon         |
| Pricing           | #0A0A0A        | #FFD60A      | Cards con borde                |
| CTA               | #0A0A0A        | #FFD60A      | Logo prominente                |

### Elementos consistentes en TODOS los slides

- Logo isotipo en esquina superior derecha (20px, color apropiado al fondo)
- Número de slide en esquina inferior derecha: Inter Regular 10px, #333333
- Margen interior: 60px en todos los lados (zona segura)
- Máximo 3 bloques de información por slide
- Máximo 40 palabras de copy por slide (excluye titulares)

### Animaciones recomendadas (si se usa Google Slides / PowerPoint)

```
Transición entre slides:  Fade, 200ms (nunca fly-in ni spin)
Aparición de elementos:   Fade in, 300ms, desde origen
Secuencia de pasos:       Aparición uno por uno, 200ms entre cada uno
```
