# Brand Guidelines — UGC Colombia

> Versión 1.0 — Uso interno y para briefing de proveedores creativos.
> Arquetipo: Creator (El Creador) + Ruler (El Soberano)
> Tagline: "UGC latino con estándar USA."

---

## 1. Identidad en una frase

UGC Colombia es una agencia boutique premium. La identidad visual comunica **autoridad sin frialdad** — editorial, sistematizada, humana. El negro y el dorado son la combinación de una pieza de lujo: no gritamos, imponemos presencia.

## 2. Uso del Logo

### 2.1 Versiones del logo (ver logo-specs.md para especificaciones técnicas)

| Versión         | Cuándo usarla                                          |
|-----------------|--------------------------------------------------------|
| Principal       | Uso general en fondos negros (#0A0A0A)                 |
| Inverso         | Sobre fondos claros (cream, blanco)                    |
| Horizontal      | Headers web, email signatures, documentos              |
| Isotipo (icono) | Favicon, perfil de redes, app icons, badges            |
| Monocromo negro | Documentos impresos, fondos dorados                    |
| Monocromo blanco| Bordados, grabado, fondos oscuros no-negro             |

### 2.2 DO — Usos correctos

- Usar el logo completo (wordmark + isotipo) sobre fondo negro con la versión dorada
- Mantener el clearspace de 2x la altura de la "U" del wordmark en todos los lados
- Escalar proporcionalmente — nunca distorsionar
- Usar la versión inversa cuando el fondo sea cream o blanco
- En fondos con foto/textura oscura: usar versión blanca o dorada
- Mínimo de uso digital: 120px de ancho para el logo horizontal completo

### 2.3 DON'T — Usos prohibidos

- No cambiar los colores del logo a colores no aprobados
- No usar el logo sobre fondos de color que no sean negro, cream, blanco o dorado
- No rotar, inclinar ni distorsionar el logo
- No añadir sombras, biselados ni efectos 3D al logo
- No colocar el logo sobre imágenes con poco contraste
- No usar versiones desactualizadas o de baja resolución
- No recrear el logo con tipografías similares pero incorrectas
- No rodear el logo con marcos o bordes no autorizados
- No yuxtaponer el logo con otros logos sin separación suficiente

## 3. Paleta de Colores — Reglas de Uso

### 3.1 Combinaciones aprobadas

**Combinación principal (90% del tiempo):**
- Fondo: `#0A0A0A` (negro editorial)
- Texto principal: `#FFFFFF`
- Acento / highlights: `#FFD60A` (dorado)
- Texto secundario: `#9CA3AF` (gray-400)

**Combinación secundaria (docs, propuestas, modo claro):**
- Fondo: `#F5F5F0` (cream)
- Texto: `#1A1A1A` (ink)
- Acento: `#FFD60A` (dorado, con moderación)
- Bordes: `#E5E5E5`

**CTA / Urgencia:**
- Usar `#DC2626` (rojo) exclusivamente para botones de acción primaria y badges de oferta
- Nunca usar el rojo como color decorativo

### 3.2 Proporciones de uso

```
Negro:  60% — fondos, bloques grandes
Blanco: 25% — texto, espacio en blanco
Dorado:  8% — highlights, acentos, CTAs secundarios
Rojo:    5% — CTAs primarios, urgencia
Crema:   2% — modo claro, docs formales
```

### 3.3 Fondos con foto

Cuando se use una imagen de fondo:
- Añadir overlay negro con opacidad 60–80% antes de colocar texto
- Mantener el texto en blanco (`#FFFFFF`)
- El acento dorado puede usarse para headlines sobre el overlay

## 4. Tipografía — Reglas de Uso

### 4.1 Jerarquía tipográfica

```
Nivel 1 — Display (Fraunces, Italic, 800-900):
  Úsalo en: Hero headline, quote de apertura, stat grande
  Color: #FFFFFF o #FFD60A sobre negro
  Ejemplo: "UGC latino con estándar USA."

Nivel 2 — H1/H2 (Fraunces, Regular o Bold, 700-800):
  Úsalo en: Títulos de sección, encabezados de página
  Color: #FFFFFF sobre negro / #1A1A1A sobre cream

Nivel 3 — H3/H4 (Inter, SemiBold, 600):
  Úsalo en: Subtítulos de módulo, card headlines
  Color: #FFFFFF o #FFD60A (énfasis)

Nivel 4 — Body (Inter, Regular, 400):
  Úsalo en: Párrafos, descripciones, UI text
  Color: #FFFFFF / #737373 (muted) sobre negro

Nivel 5 — Label/Caption (Inter, Medium, 500, ALL CAPS):
  Úsalo en: Tags, overlines, metadata, badges
  Letter-spacing: 0.1em — 0.2em
  Color: #FFD60A (sobre negro) / #4D4D4D (sobre cream)
```

### 4.2 Pairings aprobados

| Display (Fraunces)               | Body (Inter)             | Contexto              |
|----------------------------------|--------------------------|-----------------------|
| "Contenido que convierte."       | Párrafo descriptivo      | Hero / landing        |
| "133 marcas. Un sistema."        | Bullet de beneficios     | Sección de prueba     |
| "El talento ya estaba aquí."     | Descripción de servicio  | About / story         |
| "ROAS 4.2x en 60 días."         | Caso de estudio          | Social proof          |

### 4.3 Reglas adicionales

- **Nunca** mezclar más de 2 familias tipográficas en una misma pieza
- El serif (Fraunces) es siempre el protagonista — no lo uses para body text
- ALL CAPS exclusivamente en labels de máximo 4 palabras
- Mínimo de contraste: ratio 4.5:1 para texto normal (WCAG AA)
- No usar negrita en párrafos corridos — reservar para palabras clave

## 5. Estilo de Imagen

### 5.1 Tipos de imágenes aprobadas

**Fotografía documental de producción:**
- Sets reales de grabación en Colombia
- Behind-the-scenes de sesiones con creadores
- Espacios de trabajo auténticos
- Luz: natural, golden hour, contraste medio-alto
- Tratamiento: leve desaturación en sombras, conservar calidez en luces

**Retratos de creadores:**
- Luz natural de ventana o exterior
- Fondo neutro o urbano con contexto
- Expresión: enfocada, natural, no posada
- Paleta: colores de piel naturales, sin filtros agresivos

**Capturas de UI (Kreoon):**
- Dashboard screenshots limpios
- Anotaciones en dorado cuando sea necesario
- No mockups genéricos — siempre la plataforma real

### 5.2 Imágenes prohibidas

- Stock photos corporativos (handshakes, laptops en cafe, grupos sonriendo)
- Mockups de dispositivos genéricos de Dribbble
- Imágenes con texto de marca de terceros visible
- Filtros Instagram / LUTs agresivos que pierdan la paleta de marca
- Emojis como recurso visual principal

### 5.3 Tratamiento fotográfico

```
Tono general:    Cálido-oscuro, editorial
Temperatura:     Ligeramente cálida (no fría/azulada)
Contraste:       Medio-alto
Sombras:         Negras profundas (crush de sombras)
Luces:           Conservadas, sin quemar
Saturación:      -10 a -20% en tonos generales
Piel:            Natural, sin retoque excesivo
```

## 7. Iconografía y Elementos Gráficos

### Sistema de íconos
- Estilo: outline, 1.5–2px stroke, esquinas cuadradas (no redondeadas)
- Biblioteca recomendada: Lucide Icons o Heroicons Outline
- Tamaños: 16px, 20px, 24px, 32px
- Color: `#FFFFFF` base / `#FFD60A` para íconos activos o de acento

### Elementos gráficos de marca

**Línea separadora dorada:** `1px solid #FFD60A` — usada como acento editorial
**Corchete angular dorado:** [ ] — para destacar stats y claims en piezas visuales
**Punto dorado:** • — separador de listas inline en copywriting
**Grid de puntos:** patrón sutil en gris oscuro para fondos texturizados

### Formas
- Preferencia por formas rectangulares y líneas rectas (ángulo de marca)
- Evitar formas orgánicas/curvas como elementos de composición
- Los círculos se reservan exclusivamente para avatares de personas

## 8. Aplicaciones Específicas

### Web
- Fondo base: `#0A0A0A`
- Secciones alternadas: `#111111` (gray-900)
- Separador visual: línea de 1px en `#333333`
- Links: `#FFD60A` hover con transición 150ms
- Botón CTA primario: fondo `#DC2626`, texto blanco, hover `#B91C1C`
- Botón secundario: borde `1px #FFD60A`, texto `#FFD60A`, fondo transparente

### Documentos / Propuestas
- Portada: fondo negro, logo dorado, headline en Fraunces blanco
- Interior: fondo cream (`#F5F5F0`), texto ink, acentos dorados en titulares
- Tablas: header en `#1F1F1F`, texto blanco, filas alternas en `#F0F0EB`

### Redes Sociales
- Siempre incluir logo o isotipo en esquina definida (ver social-templates-spec.md)
- Texto siempre sobre overlay o zona clara — nunca texto flotante sobre textura
- Dorado como color de acento en máximo 1 elemento por pieza
