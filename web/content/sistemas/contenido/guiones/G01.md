# G01 — El problema con tu UGC

**Tipo:** Educativo — Framework
**Pilar:** Data & Angulos (TOFU-MOFU)
**Duracion:** 60 segundos
**Plataformas:** TikTok / Reels / Shorts
**Estilo:** Talking head con overlays de texto
**Voz:** Cristian Sanchez
**Musica:** Lo-fi hip hop minimalista, 85 BPM, sin drops pronunciados
**Hook:** Pain Hook

- Tono: seguro, no apresurado. No vendedor.
- `<prosody rate="95%">` para todo el guion — voz paisa naturalmente da cadencia correcta pero controlar velocidad.
- Pausas criticas marcadas con `<break>` en el script.
- `<emphasis level="strong">` en: "estrategia", "el sistema", "AURA".

**Retencion loop:** Termina con "primera consulta gratis" — genera accion inmediata. Si el viewer no convierte, el framework AURA queda en mente (brand recall).
