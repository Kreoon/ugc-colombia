Sans-serif condensada de impacto. Titulares en mayúsculas con personalidad fuerte. Único peso disponible: 400. El peso visual se logra con tamaño y contraste, nunca con bold.

Marca aplicada.

Principal · Sobre negro

Efecto Shine (implementado en web)

animation: logo-shine-sweep 5s ease-in-out infinite;
background: linear-gradient(115deg, transparent 30%, rgba(255,255,255,0.15) 44%, rgba(249,179,52,0.55) 48%, rgba(255,255,255,0.9) 50%, rgba(249,179,52,0.55) 52%, rgba(255,255,255,0.15) 56%, transparent 70%);
mix-blend-mode: screen;

Barrido dorado que cruza el logo cada 5s. Da premium sin ser intrusivo. Solo en pantalla — nunca en impresiones.

DO
  • Usar logo-dark-bg.png sobre fondo negro (#000)
  • Mantener clearspace = altura del isotipo
  • Escalar siempre proporcionalmente
  • Mínimo digital: 120px de ancho
  • Usar efecto shine solo en web/video
DON'T
  • No rotar, inclinar ni distorsionar
  • No añadir sombras, biseles ni 3D
  • No cambiar colores del logo
  • No usar sobre fondos sin contraste
  • No recrear con tipografías similares

08 · Efectos

Movimiento y presencia.

La marca se siente viva. Motion sutil, glows doradas, transiciones de 600–700ms con easing [0.22, 1, 0.36, 1].

Glow Pulse · CTAs premium

Animation keyframe

@keyframes glow-pulse {
  0%,100% { box-shadow: 0 0 8px rgba(212,160,23,0.4); }
  50% { box-shadow: 0 0 20px 4px rgba(212,160,23,0.7); }
}

Duración: 2.5s · Loop: infinite · Easing: ease-in-out

Text Reveal · Entrada de headlines

Stagger fade-up

initial: { opacity: 0, y: 32 }
animate: { opacity: 1, y: 0 }
transition: { delay: i * 0.12, duration: 0.65, ease: [0.22, 1, 0.36, 1] }

Framer Motion · stagger children · entrada premium de cada línea con 120ms de delay.

Marquee · Logos de marcas

Infinite horizontal scroll

@keyframes marquee {
  0% { transform: translateX(0%); }
  100% { transform: translateX(-50%); }
}
animation: marquee 40s linear infinite;

Logo Shine · Barrido premium

Sweep 115° con mask

mix-blend-mode: screen;
animation: logo-shine-sweep 5s ease-in-out infinite;
mask-image: url("/brand/logo-dark-bg.png");

12 · Checklist

ANTES DE PUBLICAR.

Checklist obligatorio. Si falla una casilla, no se publica.

Contenido

Hook de 1.5s brutal en frame 1
Captions activados y sincronizados
Pattern interrupts cada 2-3s
Densidad alta (sin relleno)
CTA específico y accionable
Loop perfecto (cierre conecta con inicio)
Copy con hook pre-fold
Clasificado en 1 pilar de contenido

Técnico y visual

Resolución correcta por plataforma
Audio claro, nivel consistente
Paleta: máx 3 colores visibles
Fuentes: solo Anton + Inter
Handle o URL visible
Hashtags optimizados (5-10 máx)
Horario óptimo por audiencia
UTM configurado si hay link

Reglas innegociables

× Jamás compartir métricas o nombres de clientes sin permiso

× Jamás atacar a un competidor por nombre

× Jamás mentir o exagerar datos

× Jamás publicar sin captions activos

× Jamás usar stock footage genérico

× Jamás mezclar más de 2 tipografías

× Jamás publicar contenido político

× Jamás comprar engagement o seguidores
