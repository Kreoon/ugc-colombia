**Conexion de serie:**
Este guion y el Guion 18 forman la serie "Mitos del UGC". Publicar con intervalo de 5-7 dias. El CTA de cada uno promete "el siguiente mito" — instala habito de seguimiento.

**Diferenciador vs Guion 13 (lote 2):**
El Guion 13 ataca el A/B testing mal hecho. Este ataca el volumen de creators como solucion falsa. Complementarios pero diferentes — uno habla de metodologia de testeo, este habla de estructura de campana.

**Retencion loop:** Serie instalada. El viewer sabe que hay un proximo mito y espera el CTA del siguiente video.
