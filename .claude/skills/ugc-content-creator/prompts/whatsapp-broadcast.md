# Prompt — WhatsApp Broadcast (imagen 1080x1080 + copy corto)

Aplicar cuando Tanya pida: "broadcast de WhatsApp sobre X", "mensaje para grupo de creadores", "anuncio WA Tier A".

## Contexto

Tanya maneja 3 grupos + broadcast:
- **Grupo Tier A** (creadores top — conversacion alta).
- **Grupo Tier B/C** (creadores estandar — anuncios).
- **Broadcast de Anuncios** (todos los creadores).
- **Prospects / leads** (clientes potenciales via formulario/DM).

## Workflow

1. **Definir audiencia** (creadores Tier A / Tier B-C / clientes / mixto).
2. **Definir objetivo** (info, convocatoria, anuncio de lanzamiento, recordatorio).
3. **Redactar copy** corto (max 4 lineas).
4. **Generar imagen** 1080x1080 **si aplica** (no todo broadcast necesita imagen — a veces es mejor solo texto).
5. **Proponer** CTA especifico y horario de envio.

## Reglas de WhatsApp broadcast

- **Sin spam:** maximo 2-3 broadcasts por semana a la misma lista.
- **Personalizacion:** nunca "hola familia UGC". Si hay campo de nombre, usarlo.
- **Copy corto:** 3-4 lineas max. WA no es Instagram.
- **CTA directo:** un link, un numero, una accion clara.
- **Sin cadenas:** jamas "envia esto a 10 personas".
- **Imagen opcional:** solo si agrega valor visual. Muchos broadcasts funcionan mejor text-only.

## Anatomia del copy

### Tipo A — Anuncio/Lanzamiento

```
🚀 [Titular corto — 6-8 palabras]

[1-2 lineas explicando que es y por que importa]

[CTA con link o accion]

— Equipo UGC Colombia
```

### Tipo B — Convocatoria / Brief disponible

```
📋 Brief disponible: [Marca o industria]

Buscamos creador(a) con perfil:
- [Caracteristica 1]
- [Caracteristica 2]
- [Ubicacion / idioma si aplica]

Responde con "INTERESA" si te encaja.
Deadline: [fecha].

— Equipo UGC Colombia
```

### Tipo C — Resultados / Celebracion (Tier A)

```
💥 [Creador] acaba de cerrar mes con [resultado].

[1 linea de contexto]

Si quieres ver el caso completo: [link interno]

Sigamos rompiendola en [mes proximo].

— Equipo UGC Colombia
```

### Tipo D — Recordatorio educativo (masterclass, reto)

```
🎯 Recordatorio: [evento / reto] empieza [fecha/hora].

[1 linea de beneficio]

Link para unirte: [link]

¿Preguntas? Responde este mensaje.

— Equipo UGC Colombia
```

## Imagen 1080x1080 (si aplica)

**Cuando SI generar imagen:**
- Lanzamiento de iniciativa/reto.
- Masterclass o evento en vivo.
- Celebracion visual (hito, resultado).
- Brief de nicho muy especifico (el visual ayuda a entender).

**Cuando NO generar imagen:**
- Mensajes operativos cortos.
- Recordatorios.
- Anuncios de logistica.

**Prompt Nanobanana ejemplo (lanzamiento reto):**
```
scene: square 1:1 graphic composition for WhatsApp broadcast,
bold editorial layout with dark #000000 top half and cream #F5F5F0 bottom half,
centered minimal icon or number in amber #F9B334 at the intersection,
documentary photograph fragment in the top half showing a Colombian creator at work,
generous negative space reserved for typography overlay in the bottom half,
matte grain, premium editorial boutique feel,
no text, no logos, no readable elements.
```

**Estructura visual:**
- **Top 50%:** foto o visual editorial.
- **Bottom 50%:** fondo crema con espacio para titular (Anton 48-60px).
- **Acento amarillo:** un icono o palabra clave.
- **Logo UGC Colombia:** bottom-right, version negra sobre cuadro blanco pequeno.

## CTA por tipo de audiencia

| Audiencia        | CTA tipico                                              |
|------------------|---------------------------------------------------------|
| Creadores Tier A | "Responde INTERESA", link a brief, link a form          |
| Creadores B/C    | Link a form, link a evento, numero para llamar          |
| Clientes/prospects | Link a agendar call, link a propuesta, link a Kreoon |
| Mixto broadcast  | Link a contenido publico (post, reel, carrusel)         |

## Horarios recomendados

- **Lunes 9-11am:** planificacion semanal, anuncios.
- **Martes-Jueves 2-4pm:** briefs, convocatorias.
- **Viernes 9-10am:** resultados, celebracion.
- **Evitar:** sabado, domingo, despues de 7pm (rebote alto).

## Checklist

- [ ] Copy max 4 lineas.
- [ ] 1 emoji funcional al inicio (nunca mas de 2 en total).
- [ ] CTA claro y unico.
- [ ] Link acortado si tiene UTM (usar `lnk.ugccolombia.co` si existe, o bit.ly).
- [ ] Firma "— Equipo UGC Colombia" al final.
- [ ] Si hay imagen: formato JPG 1080x1080, compresion OK para WA.
- [ ] Audiencia correcta (no mezclar clientes con creadores).
- [ ] Horario apropiado.
