## 1. North Star Metric (NSM)

### NSM: **MRR Activo Cobrado en USD**

Una sola métrica que captura salud comercial, retención y cobranza efectiva. No vale "vendido", vale "cobrado y activo".

### Árbol de métricas derivadas

```
                    MRR ACTIVO COBRADO (NSM)
                            |
        ┌───────────────────┼───────────────────┐
        |                   |                   |
   NUEVO MRR          MRR EXPANSION         MRR PERDIDO (-)
   (new logos)        (upsells)             (churn)
        |                   |                   |
   ┌────┴────┐         ┌────┴────┐         ┌────┴────┐
   Leads     Close     Upsell    NPS       Logo      Revenue
   Calificados Rate    Rate      Score     Churn     Churn
   |          |        |         |         |         |
   CAC        Demos    QBRs      Health    Renewal   Downgrade
   Source     Hechas   Trigger   Score     Rate      Rate
        |
   ┌────┴────┐
   Tráfico   Conv. Lead
   (IG, web, (form/llamada)
    referral)
```

### Métricas espejo (counter-metrics para evitar gaming)

- **Gross Margin %** (no quemar plata cerrando barato)
- **Delivery Health Score** (1-10) — calidad de entregables por cliente
- **Creator NPS** — salud del lado de oferta (creadores)
